  配色方案

  主色调（蓝色系）：
  - #3b82f6 - 主蓝色（品牌色）
  - #1d4ed8 - 深蓝色
  - #1e40af - 更深的蓝色
  - #eff6ff / #dbeafe / #e0f2fe - 蓝色渐变背景

  辅助色（橙色系）：
  - #f97316 - 橙色
  - #fb923c - 浅橙色
  - #c2410c - 深橙色
  - #fff7ed / #ffedd5 - 橙色渐变背景

  绿色：
  - #22c55e / #16a34a - 成功/活跃状态

  中性色：
  - #0f172a / #1e293b - 主文字色
  - #334155 / #475569 - 次要文字
  - #64748b / #94a3b8 - 辅助文字/占位
  - #f8fafc / #f1f5f9 - 卡片背景
  - #ffffff - 纯白背景
  - #e2e8f0 / #cbd5e1 - 边框/分割线

  ---
  视觉风格

  卡片设计：
  - 圆角：26px（大卡片）、18px（中）、14-16px（小）
  - 背景：rgba(255, 255, 255, 0.95) + 毛玻璃效果 backdrop-filter: blur(20px)
  - 边框：1px solid rgba(226, 232, 240, 0.8)
  - 阴影：多层柔和阴影
  box-shadow:
    0 8px 32px rgba(0, 0, 0, 0.08),
    0 2px 8px rgba(0, 0, 0, 0.04),
    inset 0 1px 0 rgba(255, 255, 255, 0.9);

  按钮设计：
  - 默认：白色背景 + 细边框
  - 激活：蓝紫渐变 linear-gradient(135deg, #1e40af 0%, #2563eb 100%)
  - 圆角：24px（标签页）、12-14px（按钮）

  背景效果：
  - 页面背景：从上到下的浅灰渐变 + 径向光效
  - 装饰光斑：用 ::before 伪元素 + radial-gradient 实现

  ---
  排版

  字体层级：
  - 大数字：64px / font-weight: 900 / letter-spacing: -2px
  - 中数字：28px / font-weight: 800
  - 主标题：20-22px / font-weight: 700-800
  - 副标题：16px / font-weight: 600-700
  - 正文：13-15px / font-weight: 500-600
  - 辅助文字：11-12px / font-weight: 500

  ---
  动画与交互

  过渡动画：
  transition: all 0.2s ease;

  悬停效果：
  - 向上位移 translateY(-1px/-2px)
  - 阴影增强
  - 背景色微变

  脉冲动画：
  @keyframes pulse {
    0%, 100% { opacity: 1; transform: scale(1); }
    50% { opacity: 0.6; transform: scale(1.2); }
  }

  @keyframes ping {
    75%, 100% { transform: translate(-50%, -50%) scale(2.5); opacity: 0; }
  }

  ---
  布局规范

  间距系统：
  - 小间距：6-10px
  - 中间距：14-18px
  - 大间距：24-28px
  - 超大间距：32-36px

  网格布局：
  - 侧边栏宽度：240px
  - 顶部卡片：380px + 1fr
  - 底部图表：2fr + 1fr

  ---
  图表样式（ECharts）

  折线图：
  - 线条宽度：3px
  - 平滑度：0.4
  - 区域渐变：从 18-20% 透明度降到 2%
  - 数据点：圆形，白色描边 2px

  柱状图：
  - 柱体圆角：8px（顶部）
  - 柱体渐变：双色垂直渐变
  - 堆叠布局

  通用配置：
  - 网格线：虚线 #f1f5f9
  - 坐标轴文字：#94a3b8，11-13px
  - Tooltip：半透明白色 + 圆角 12 + 阴影

  ---
  这个设计风格属于 现代 B 端数据分析仪表盘风格，特点是：清爽、专业、有呼吸感、微渐变 + 毛玻璃效果。
