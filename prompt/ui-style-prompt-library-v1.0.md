## 📚 风格列表
### 核心风格
#### 1. 平面设计系统（Flat UI v1.0）
- **描述**：简洁高效的扁平系统，无阴影或渐变，强调内容优先和简洁美学。通过极简布局、清晰排版和必要元素的精心选择创造现代化界面。
- **标签**：当代流行、扁平化、极简主义、粗犷大胆、多彩缤纷、简洁、电商零售、design-system
- **适用场景**：当代应用、用户友好的界面，体现"少即是多"的哲学

#### 2. Material Design 系统
- **描述**：完整的Material Design系统指南，展示设计原则、组件和交互模式。提供现代化的卡片布局、阴影阶层和涟漪反馈效果。
- **标签**：当代流行、立体深度、几何规整、企业应用、粗犷大胆、多彩缤纷
- **适用场景**：安卓应用、跨平台企业级应用

#### 3. 滚动叙事
- **描述**：滚动驱动的叙事，采用视差效果和滚动互动讲述沉浸式故事。通过动态视觉和互动元素创造引人入胜的用户体验。
- **标签**：当代流行、动效驱动、沉浸式、创意展示、内容媒体
- **适用场景**：叙事驱动的内容、创意展示类网站

#### 4. Fluent Design 2.0
- **描述**：微软最新系统，展现Acrylic和Mica材质的现代玻璃态效果。融合当代美学与功能设计，创造精致洗练的用户体验。
- **标签**：当代流行、半透明、立体深度、企业应用、现代
- **适用场景**：Windows应用、现代企业级系统

#### 5. 极简主义
- **描述**："少即是多"哲学，强调留白、简洁排版和本质元素。透过极简风格和简洁构图建立优雅界面。
- **标签**：当代流行、极简主义、扁平化、创意展示、简洁
- **适用场景**：内容类应用、极简风格产品

#### 6. 拟物化
- **描述**：模拟真实世界的纹理、阴影和材质创造逼真数字界面。使用深色调、阴影和3D效果增强视觉深度。
- **标签**：经典、立体深度、创意展示、企业应用、纹理
- **适用场景**：需要直观操作和触觉反馈的应用

### 视觉元素类风格
#### 1. 排版
- **描述**：排版主导的风格，文字本身的视觉表现传达信息和情感。强调排版的表现力和设计性，创造引人注目的视觉语言。
- **标签**：当代流行、编辑、创意展示、粗犷大胆
- **适用场景**：内容类应用、媒体平台

#### 2. 无障碍设计
- **描述**：无障碍UI风格，强调色彩对比、清晰视觉级和键盘友好的互动。创造包容性设计，确保所有用户都能便捷使用。
- **标签**：当代流行、无障碍、high-contrast、企业应用
- **适用场景**：公共服务类应用、企业级系统

#### 3. 反设计工作室
- **描述**：大胆实验的风格，打破设计规则并拒绝既定惯例。创造富有挑战性的设计，展现叛逆创意和非传统美学。
- **标签**：当代流行、粗犷大胆、实验性、多彩缤纷、创意展示
- **适用场景**：创意类项目、艺术展示平台

#### 4. 亲生物设计
- **描述**：Biophilic Design是亲生物设计风格，适合健康、环保和生活方式类应用。将自然元素融入数字界面，使用绿色调、有机形状、流动曲线和自然纹理。
- **标签**：当代流行、有机流动、多彩缤纷、自然、和谐、创意展示
- **适用场景**：健康、环保、生活方式类应用

#### 5. 黏土质感
- **描述**：Claymorphism是3D黏土质感设计风格，适合创意设计、插画风格和儿童产品。通过多层内外阴影、柔和的色彩渐变和圆润的造型，创造如同真实黏土般的效果。
- **标签**：当代流行、立体深度、柔和、多彩缤纷、趣味、创意展示、电商零售
- **适用场景**：创意设计、儿童产品、插画类应用

#### 6. 深色模式
- **描述**：深色模式，采用深色调和高对比度创造舒适视觉。提供眼睛友好的深色美学和精细的视觉平衡。
- **标签**：当代流行、深色、极简主义、现代
- **适用场景**：所有应用的深色主题适配

#### 7. 织物纹理
- **描述**：织物纹理风格，采用布料纹理和交错效果。创造触觉感和真实感，展现织物材质特性和视觉质感。
- **标签**：当代流行、有机流动、多彩缤纷、纹理、创意展示
- **适用场景**：时尚、家居类应用

#### 8. Glassmorphism（玻璃态）
- **描述**：玻璃态风格，采用毛玻璃效果和背景模糊。创造现代透明界面、精致视觉级和浮动感视觉体验。
- **标签**：当代流行、半透明、立体深度、渐变、现代、创意展示
- **适用场景**：现代科技应用、后台管理系统

#### 9. Gradients 渐变
- **描述**：充满活力的渐变风格，采用平滑色彩过渡和现代美学。提供色彩动态演变、视觉深度和流畅视觉过渡。
- **标签**：当代流行、多彩缤纷、渐变、创意展示
- **适用场景**：各类需要视觉增强的界面

#### 10. 手绘涂鸦
- **描述**：通过手写字体、SVG滤镜和随性线条创造温暖亲切的视觉体验。结合粘贴便签、网格纸背景和手绘边框元素，营造友好轻松的设计氛围。
- **标签**：永恒、趣味、有机流动、手写、创意展示、非正式
- **适用场景**：笔记类应用、创意工具、教育类产品

#### 11. 全息箔效果
- **描述**：全息箔纸效果，采用彩虹色和虹彩色调。创造反射表面特性、彩虹色位移效果和光学变彩感。

- **适用场景**：创意类、科技类应用

#### 12. Ink Wash（水墨）
- **描述**：传统中国水墨画启发的风格，采用水墨渐变和有意义的构图。结合传统美学与当代数位设计、东方美感。
- **标签**：chinese、传统、艺术、zen、minimalist
- **适用场景**：文化类、艺术类应用，东方风格产品

#### 13. 皮革质感
- **描述**：深色皮革面板风格，采用径向渐变和皱纹纹理。通过皮革特性营造厚重奢华的视觉体验和高级感。

- **适用场景**：高端产品、奢侈品相关应用

#### 14. Liminal Space 边界空间

## 🎯 使用方法
1. 选择适合的风格，复制风格名称+描述+标签作为提示词的一部分
2. 结合具体页面需求，比如「设计一个数据概览后台页面，采用亲生物设计风格，有机形态，柔和渐变」
3. 可组合多种风格，比如「水墨风格+极简主义，设计一个文化类APP的个人中心页面」
