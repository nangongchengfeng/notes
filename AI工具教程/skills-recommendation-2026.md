# 🛠️ OpenClaw 精选技能推荐（2026最新）
本文精选OpenClaw生态中高价值、高使用率的核心技能，覆盖编码增强、规划能力、开发体验、界面设计等多个场景，所有技能均经过验证可用，可直接通过命令安装。

---

## ⭐ 核心必装技能
### 1. everything-claude-code | 核心编码超能力
**能力说明**：Claude Code 原生编码能力增强，支持多语言全栈开发、代码理解、重构、调试一站式处理，是编码场景的核心基础技能
**适用场景**：全栈开发、代码重构、Bug排查、功能实现
**安装命令**：
```bash
/plugin marketplace add nextlevelbuilder/everything-claude-code
/plugin install everything-claude-code@everything-claude-code
```

### 2. superpowers | 能力增强包
**能力说明**：扩展Agent基础能力边界，新增多工具联动、上下文感知增强、长任务自动拆解等高级能力，大幅提升复杂任务处理效率
**适用场景**：复杂多步骤任务处理、跨工具联动需求、长周期项目管理
**安装命令**：
```bash
/plugin marketplace add nextlevelbuilder/superpowers
/plugin install superpowers@superpowers
```

### 3. planning-with-files | 智能文件规划
**能力说明**：基于项目需求自动生成文件结构、目录规划、代码拆分方案，支持多项目类型（前端/后端/工具/文档）的结构自动设计
**适用场景**：新项目初始化、项目架构设计、代码重构目录调整
**安装命令**：
```bash
/plugin marketplace add nextlevelbuilder/planning-with-files
/plugin install planning-with-files@planning-with-files
```

### 4. gopls-lsp | Go语言服务增强
**能力说明**：集成Go官方LSP服务，提供Go代码智能补全、语法检查、跳转定义、重构建议等专业级Go开发支持
**适用场景**：Go语言项目开发、Go性能优化、Go代码质量检查
**安装命令**：
```bash
/plugin marketplace add nextlevelbuilder/gopls-lsp
/plugin install gopls-lsp@gopls-lsp
```

### 5. claude-hud | 开发体验优化
**能力说明**：优化Agent交互体验，新增任务进度可视化、实时状态反馈、快捷操作面板等功能，大幅降低交互成本
**适用场景**：长任务监控、高频交互场景、开发效率提升
**安装命令**：
```bash
/plugin marketplace add nextlevelbuilder/claude-hud
/plugin install claude-hud@claude-hud
```


## 📌 安装说明
1. 所有命令均在OpenClaw控制台直接执行即可，无需额外配置
2. 安装完成后可通过 `/skills list` 命令查看已安装的技能列表
3. 技能更新可通过 `/plugin update <技能名>` 命令一键升级
4. 如安装失败可先执行 `/plugin marketplace refresh` 刷新插件市场缓存后重试
